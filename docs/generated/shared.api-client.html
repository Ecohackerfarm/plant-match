Namespace: api-client

shared. api-client

API client based on axios Every api call should be able to call like this: functionName({parameter}) => Promise e.g.: getCropsByName( { name : "foo", index : 0, length : 0, } ) In addition to API calls there are functions that configure the axios HTTP client.
